# GoalChain

## 소개 (Overview)

GoalChain은 목표를 **체인(Chain) 구조로 연결하여 관리하는 목표 관리 서비스**입니다.
사용자는 목표를 단순한 할 일 목록이 아니라 **연결된 흐름**으로 관리할 수 있습니다.

작은 목표들을 하나씩 달성하며 체인을 이어가면
자연스럽게 더 큰 목표로 이어지는 구조를 제공합니다.

핵심 아이디어는 다음과 같습니다.

> 작은 목표들이 연결되어 큰 성취를 만든다.

---

## 문제 정의 (Problem)

기존의 많은 생산성 앱은 할 일 목록 중심으로 구성되어 있습니다.
하지만 이런 방식에는 몇 가지 문제가 있습니다.

* 목표들이 서로 연결되어 있지 않음
* 진행 상황을 직관적으로 보기 어려움
* 작은 성취가 쉽게 잊혀짐
* 장기적인 동기 유지가 어려움

이 때문에 많은 사용자들이 목표를 세워도 중간에 포기하게 됩니다.

---

## 해결 방법 (Solution)

GoalChain은 목표를 **체인 형태로 연결하여 시각화**합니다.

사용자는 다음과 같은 흐름으로 목표를 관리합니다.

1. 목표 생성
2. 작은 목표들을 연결
3. 목표 달성 시 체인 확장
4. 전체 목표 진행 흐름 확인

이 구조를 통해 사용자는 **자신의 성취 흐름을 시각적으로 확인**할 수 있습니다.

---

## 주요 기능 (Core Features)

### 1. 목표 체인 생성

사용자는 목표를 카드 형태로 생성하고 서로 연결할 수 있습니다.

### 2. 목표 진행 시각화

목표 달성 과정이 체인 형태로 표시되어 진행 상황을 직관적으로 확인할 수 있습니다.

### 3. 목표 카드 관리

각 목표에는 다음 정보가 포함될 수 있습니다.

* 목표 제목
* 설명
* 진행 상태
* 연결된 다음 목표

### 4. 목표 흐름 확인

사용자는 하나의 큰 목표로 이어지는 **전체 목표 체인**을 확인할 수 있습니다.

## 실행 방법 (Getting Started)

### 1. 저장소 클론

```
git clone <repository-url>
```

### 2. 의존성 설치

```
npm install
```

### 3. 개발 서버 실행

```
npm run dev
```

브라우저에서 아래 주소로 접속합니다.

```
http://localhost:5173
```
